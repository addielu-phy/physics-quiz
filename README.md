# 理化自學評量網頁（國二下期末）

學生可登錄姓名、線上作答整份試卷，系統會**自動評分**、做**錯題與單元分析並附詳解**，並支援**第二次、第三次練習**（含「只練錯題」）。成績紀錄存在使用者的瀏覽器（localStorage）。

- 本卷：高雄市立五福國民中學 113 學年度第 2 學期 二年級 第 3 次段考（自然／理化），50 題、滿分 100。
- 題目來源：學校官方公開之歷屆試題庫，僅供同學自學練習使用。

## 檔案結構
- `index.html`：頁面外殼
- `style.css`：深色主題樣式
- `data.js`：題庫（題目、選項、正解、單元分類、詳解）
- `app.js`：登錄／作答／評分／分析／重複練習邏輯
- `assets/`：各題附圖（自試卷擷取）

## 更換題目
編輯 `data.js` 中的 `QUESTIONS` 陣列即可：每題包含題號 `id`、單元 `u`、題目 `s`、選項 `o`、正解索引 `a`（0=A）、詳解 `e`、附圖 `img`。

## 本機預覽
```
python -m http.server 8731 --directory .
```
然後開啟 http://localhost:8731
